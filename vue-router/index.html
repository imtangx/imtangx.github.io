# 什么是路由？

传统的服务器路由，在点击页面链接时，会根据链接跳转到新的 html。而这里的路由指的是客户端路由，即 Javascript 可以拦截当前页面的跳转请求，动态获取更新数据，并在无需重新加载的清空下更新当前页面，让用户的体验更加丝滑。说直白就是页面内跳转。

# 带参数的动态路由匹配
